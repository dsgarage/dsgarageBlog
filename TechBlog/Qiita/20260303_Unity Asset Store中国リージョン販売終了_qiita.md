# Unity Asset Store 中国リージョンパブリッシャー販売終了 — 2026年3月31日

## はじめに

2026年3月3日、Unity Asset Store チームから重要なアナウンスがありました。

**2026年3月31日をもって、中華圏（香港・マカオを含む）のパブリッシャーによるアセットが、グローバルのUnity Asset Storeでの販売を終了します。**

対象は **623社のパブリッシャー、2,668個のアセット**です。

## 何が起きるのか

### 変わること

| 項目 | 3月31日以前 | 3月31日以降 |
|:---|:---|:---|
| 新規購入 | 可能 | **不可** |
| パブリッシャーによるアップデート | あり | **なし** |
| パブリッシャーによるサポート | あり | **なし** |

### 変わらないこと

| 項目 | 状態 |
|:---|:---|
| 購入済みアセットの利用 | **継続可能** |
| My Assets からのダウンロード | **継続可能** |
| Package Manager 経由のダウンロード | **継続可能** |
| ライセンスの有効性 | **有効のまま** |
