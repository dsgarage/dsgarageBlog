## 5人のスピーカーの統合メッセージ

1. **Ian Schreiber**（理論）: 認知負荷を管理しろ — ワーキングメモリのスロット数を意識し、Extraneous Load を最小化、Germane Load を適切にコントロール
2. **Lauryn Ash**（フレームワーク）: 何をいつ教えるかをトリアージしろ — Tutorial Matrix（DO / SHOW / REINFORCE / CUT）で意思決定
3. **Monica Fan**（結果と警鐘）: さもなくばプレイヤーは離脱する — 時間のない大人ゲーマーをバーンアウトさせるな。Ask less, Give more, Go deeper
4. **John Ryan**（実践・ナラティブ）: Narrative Coherence で物語を伝えろ — 6つのケーススタディから学ぶ moment-to-moment vs safe space の使い分け。プレイヤーの邪魔をせずコンテキストを提供
5. **Michael Jones**（実践・オーディオ）: 音で導け — 4 Circles モデル（Medium → Intermediate → Immediate → Long form）でテキストやUIに頼らずプレイヤーを自然に誘導
