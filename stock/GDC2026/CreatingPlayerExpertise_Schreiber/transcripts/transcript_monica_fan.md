| # | Clippy のセリフ | 問題点 |

| #35 | "Welcome to exciting AAA Game!" | 似たゲームをプレイ済みでもスキップできない |

### チュートリアル + カットシーン地獄（#40）
- チュートリアルとカットシーンをようやく終えた
- その時点で**就寝時間を1.5時間オーバー**
- 明日のプレゼンのために仕方なく寝る

### 離脱パターン（#41）
- ゲームを閉じて **Balatro を1ラウンド**、または軽いTV番組を見る
- 翌日の仕事の評価に備えて**時間通りに就寝**
- → AAA ゲームではなく、手軽なゲームに逃げる

### Possible Solution — Holistic Approach（#43）
**Skill Depth/Mastery × Skill Variety のグラフ**
- Y軸: Skill Depth / Mastery
- X軸: Skill Variety
- 左下 → 右上の楕円形の **Sweet Spot**
- Sweet Spot の外: 左下 = **Boring**、右上 = **Challenging**
- ホリスティックなアプローチで全体を見る必要がある

### Games that use more to do less（#44）
- **Roblox**, **Fortnite**, **CrazyGames**
- これらのゲームは「より多くのことをより少ない認知負荷で」実現
- プレイヤーが自然に学べる設計

### 結論（#45）
- **Tech Advancement =/= Cognitive Advancement**（技術の進歩 ≠ 認知能力の進歩）
- **Accept that most players have other things to worried about in life**（プレイヤーには人生で心配すべき他のことがある）
- **Ask less, Give more, Go deeper**（求めるものを減らし、与えるものを増やし、深く掘り下げよ）

### AAA 業界の危機（#42）
**"Last Year we saw a lot of headlines along the lines"**
- "AAA Game Industry is Dying, Ex-Nexon CEO Believes" (Dec 2025)
- "Former Sony exec says focusing on high-budget blockbusters is 'a death sentence' for the games industry"
- "GTA 6's delay doesn't mean the games industry's in trouble - it's already dead"
- "The AAA Video Game Studio is Dead. What Now?" (Chris Kaleiki, Apr 2024)

- → チュートリアル過多は AAA 離れの一因

## キーコンセプト整理

### 「木曜日シナリオ」— 大人ゲーマーのリアリティ
- 仕事・家事・育児を終えた後の限られた時間（約2時間）
- 9pm〜11pm の貴重なゲーム時間
- チュートリアル + カットシーンで**実際のゲームプレイに到達できない**

### Clippy メタファー
- AAA ゲームのチュートリアルを Microsoft Clippy に擬人化
- スキップ不可、今すぐ不要な情報の洪水、不完全な説明
- 情報過多の視覚的表現（テキスト重なりスライド）

### Game Over サイクル
1. 木曜日: チュートリアルで時間を消費 → 就寝
2. 翌週木曜日: コントロールを忘れている → 再教育なし → 死亡の繰り返し
3. ゲームを閉じて **Balatro** や TV に逃避
4. → AAA ゲームから**永久離脱**

### AAA 業界への警鐘
- **購買力最大のプレイヤー = プレイ時間最小のプレイヤー**（大人ゲーマー）
- チュートリアル過多は「あなたの時間よりコンテンツが大事」というメッセージ
- 「楽しい部分」に到達する前にバーンアウトさせている
- AAA 業界の危機ヘッドラインとの直接的関連
